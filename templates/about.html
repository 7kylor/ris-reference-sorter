About Reference Manager

Reference Manager is a modern web application designed to help researchers, students, and academics efficiently manage and format their academic references. Built with simplicity and speed in mind, it streamlines the process of organizing citations from multiple sources.

Features

  • Multiple Citation Styles: Support for APA, MLA, Chicago, Harvard, and IEEE formats
  • URL-Based Extraction: Automatically extract metadata from arXiv, DOI, PubMed, and web pages
  • RIS File Support: Upload and process Research Information Systems files
  • Duplicate Detection: Automatically identify and remove duplicate references
  • Multiple Export Formats: Export as plain text, numbered list, Markdown, BibTeX, or RIS
  • Real-time Search: Quickly find references in your collection
  • Individual Management: Copy or delete specific references

How It Works

Reference Manager processes your RIS files or extracts metadata from URLs, formats them according to your chosen citation style, and organizes them alphabetically. All processing happens in your browser session, ensuring your data remains private and secure.

Technology

Built with Flask, Python, and modern web technologies. The application uses APIs from arXiv, CrossRef, and PubMed to fetch accurate metadata, ensuring high-quality citation formatting.

Open Source

Reference Manager is open source and available under the MIT License. Contributions and feedback are welcome.
